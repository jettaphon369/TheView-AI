TheView Stock v27.8

แก้ไข 3 รายการ:
1. หน้าแรกของผู้มีสิทธิ์อนุมัติ
   - เพิ่มการ์ด “รายการรออนุมัติ” แบบเด่น
   - สถานะระบบแสดงจำนวนรายการรออนุมัติ
   - สถานะระบบแสดงจำนวนสินค้าใกล้หมด

2. เปลี่ยนคำและลำดับความสำคัญ
   - เปลี่ยน “รอตรวจ” เป็น “รออนุมัติ”
   - การ์ดรออนุมัติอยู่ด้านบนและเด่นกว่าการ์ดอื่น

3. รายละเอียดรายงานสต๊อก
   - รายการเบิกออกแสดงชื่อผู้เบิก
   - รายการรับเข้าแสดงชื่อผู้รับเข้า
   - ถ้ามีข้อมูล จะแสดงชื่อผู้อนุมัติด้วย

ไฟล์ที่ต้องอัป:
- app.js
- main.css
- index.html
- service-worker.js

ไม่ต้องแก้ Firebase Rules
