# TheView Stock v26.2 — ตำแหน่งและผู้ช่วยอนุมัติ

## ตำแหน่ง 4 ระดับ
- `staff` = พนักงาน
- `captain` = กัปตัน / ธุรการ
- `manager` = ผู้จัดการ
- `admin` = แอดมิน

## การมองเห็นคิว Approval
เฉพาะบุคคลต่อไปนี้เท่านั้นที่จะเห็นและดำเนินการในคิวตรวจ:
1. กัปตัน / ธุรการ
2. ผู้จัดการ
3. แอดมิน
4. พนักงานที่กัปตันเลือกเป็นผู้ช่วยอนุมัติ (`permissions.canApprove = true`)

พนักงานทั่วไปจะไม่เห็นเมนู Approval และ Firestore Rules จะไม่อนุญาตให้อ่านคิวโดยตรง

## การเลือกผู้ช่วยอนุมัติ
กัปตันเข้าเมนู **โปรไฟล์ → จัดการผู้ช่วยอนุมัติ** แล้วเลือกพนักงานหนึ่งคน หลายคน หรือยกเลิกทั้งหมดได้

เมื่อผู้อนุมัติคนหนึ่งอนุมัติหรือปฏิเสธ รายการจะถูกนำออกจากคิวแบบ Realtime ทำให้ผู้อื่นไม่สามารถดำเนินการซ้ำได้

## การตั้งค่าหลังอัปโหลด
1. อัปไฟล์ทั้งหมดขึ้น GitHub หรืออัปเฉพาะ `app.js`, `index.html`, `service-worker.js`
2. นำ `firestore.rules` ไปวางที่ Firebase Console → Firestore Database → Rules
3. กด Publish
4. ตั้ง `role` ของสมาชิกใน `members/{UID}` เป็นหนึ่งใน 4 ค่าข้างต้น
5. ออกจากระบบและเข้าสู่ระบบใหม่

## หมายเหตุ
- การสร้างบัญชียังคงทำผ่าน Firebase Authentication ตามแนวทางฟรี
- แอดมินจัดตำแหน่งและสถานะสมาชิกได้จากหน้าโปรไฟล์
- เวอร์ชันหน้าเว็บและ Cache คือ v26.2
